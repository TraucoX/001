Solución Técnica de Alto Nivel

Arquitectura

La aplicación seguirá una arquitectura de tres capas:

  1. Capa de Presentación: Frontend web utilizando HTML, CSS (Bulma) y JavaScript
  2. Capa de Lógica de Negocio: Backend en Python utilizando el framework Flask
  3. Capa de Datos: Base de datos PostgreSQL

Tecnologías Principales

  • Frontend:
    • HTML5
    • Bulma CSS para el diseño responsive
    • JavaScript (Vue.js) para interactividad
    • Chart.js para visualizaciones
  • Backend:
    • Python 3.9+
    • Flask (framework web)
    • SQLAlchemy (ORM)
    • Flask-RESTful para API RESTful
    • Flask-JWT-Extended para autenticación
  • Base de Datos:
    • PostgreSQL 13+
  • Despliegue:
    • Docker para containerización
    • Nginx como servidor web y proxy inverso
    • Gunicorn como servidor WSGI

Diagrama de Arquitectura

    [Usuario] <-> [Navegador Web]
                      ^
                      |
                      v
    [Nginx (Proxy Inverso)] <-> [Gunicorn + Flask App]
                                        ^
                                        |
                                        v
                                [PostgreSQL DB]

Esta arquitectura permite una separación clara de responsabilidades, escalabilidad y mantenimiento sencillo.
