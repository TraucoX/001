Historias de Usuario - Sprint 1

GAST-1: Configuración del Entorno de Desarrollo

Como desarrollador, quiero tener un entorno de desarrollo configurado para poder comenzar a trabajar en el proyecto.

Criterios de Aceptación:

  • Repositorio Git inicializado
  • Entorno virtual de Python creado
  • Dependencias iniciales instaladas (Flask, SQLAlchemy, etc.)
  • Archivo README.md con instrucciones de configuración

Estimación: 2 puntos

GAST-2: Implementar Registro de Usuarios

Como usuario nuevo, quiero poder registrarme en la aplicación para comenzar a usar el sistema de control de gastos.

Criterios de Aceptación:

  • Formulario de registro con campos: nombre, email, contraseña
  • Validación de campos (email único, contraseña segura)
  • Almacenamiento seguro de contraseñas (hashing)
  • Mensaje de confirmación tras registro exitoso

Estimación: 5 puntos

GAST-3: Implementar Login/Logout

Como usuario registrado, quiero poder iniciar y cerrar sesión en la aplicación para acceder a mis datos de forma segura.

Criterios de Aceptación:

  • Formulario de login con email y contraseña
  • Autenticación mediante JWT
  • Página de inicio personalizada después del login
  • Opción de logout visible en todas las páginas autenticadas
  • Redirección a página de login después del logout

Estimación: 5 puntos

GAST-4: Crear Estructura Básica de la Base de Datos

Como desarrollador, quiero tener una estructura inicial de la base de datos para poder comenzar a implementar las funcionalidades principales.

Criterios de Aceptación:

  • Modelo de Usuario creado
  • Modelo básico de Gasto creado (sin categorías aún)
  • Migraciones de base de datos configuradas
  • Script para inicializar la base de datos con datos de prueba

Estimación: 3 puntos

GAST-5: Diseñar Interfaz Básica con Bulma CSS

Como usuario, quiero ver una interfaz de usuario básica y responsive para poder interactuar con la aplicación en diferentes dispositivos.

Criterios de Aceptación:

  • Navbar con opciones de login/registro/logout
  • Página de inicio con descripción básica de la app
  • Formularios de login y registro estilizados con Bulma
  • Diseño responsive que funciona en móviles y escritorio

Estimación: 5 puntos
